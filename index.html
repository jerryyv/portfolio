Projects

FireChat

FireChat is an app where authenticated users can create/join multiple chat rooms and send messages in real time. The front-end was built with react while implementing CSS-in-JS using styled-components and material-ui. Google firebase was used in the back-end to handle user-authentication and data storage. React-firebase-hooks was used to fetch any real-time updates in the database and user state. The live site was deployed with firebase hosting.

Coffee Shop

Coffee Shop is a JAMStack landing page for an e-commerce coffee store. The front-end was built using next.js(for static generation) and typescript, with contenful as the content management system. It is a fully responsive app which was styled using tailwind CSS. The app also features a third-party shopping cart called Snipcart to enable payements and give it the feel of a fully functional e-commerce store. The live site was deployed on Vercel

NextNews

NextNews is an app that lets users find top trending news articles from across Canada in various categories. The data for the articles is fetched from a third-party API called News API. The front-end was developed using Next.js in order to implement features such as server-side rendering, page routing and a custom 404 page. Tailwindcss was used to provide fully responsive styling to the app. The live site was deployed on Vercel.
