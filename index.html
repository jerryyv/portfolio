FireChat

FireChat is an app where authenticated users can create/join multiple chat rooms and send messages in real time. The front-end was built with react and implemented CSS-in-JS using styled-components and material-ui components. Google's firebase was used in the back-end to handle user-authentication and data storage. React-firebase-hooks was used to fetch any real-time updates in the database and user state. The live site was deployed with firbase hosting.

NextNews

NextNews is an app that lets users find top trending news articles from across Canada in various categories. The data for the articles is fetched from a third party API called News API. Next.js was chosen to develop the front-end and was proven useful in implementing features such as server-side rendering, page routing and pagination. Tailwindcss was used to provide fully responsive styling to the app. The live site was deployed on Vercel.
